# Architecture Decision Records

This folder holds **ADRs** — short, dated notes that capture a significant
architectural or design decision, the context behind it, and its consequences.
They are append-only history: when a decision changes, add a new ADR that
supersedes the old one rather than rewriting it.

Format: lightweight [MADR](https://adr.github.io/madr/) — Status, Context,
Decision, Consequences. Number files sequentially (`NNNN-title.md`).
